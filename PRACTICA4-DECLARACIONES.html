Sentencias de JavaScript

Un programa en JavaScript es una lista de sentencias que serán ejecutadas por una computadora.

En HTML, las sentencias de JavaScript son ejecutadas por el navegador.

Las sentencias de JavaScript se separan con punto y coma.

Se permiten múltiples sentencias en una línea.

El mejor lugar para cortar una línea de código es después de un operador o una coma.

Los bloques de código en JavaScript se escriben entre { y }
